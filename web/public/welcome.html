The loop

Setup · read-only · keyless works

git clone https://github.com/MorkeethHQ/mount-helicon && cd mount-helicon && pip install -e .

helicon init              # detects Claude Code, Cursor, Obsidian, git
helicon scan              # read-only ingest
helicon rot               # the exam: 10 classes, deterministic, free
helicon watch --install   # notify only on new drift

A Qwen key (Model Studio free tier) adds live-judged contradiction and grounding verdicts. Without one, every deterministic check still runs. Your files are never written. Access model · The rot catalogue
